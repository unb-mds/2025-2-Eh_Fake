# v 0.2.0  
**Data de Lançamento:** 02/12/2025  

## Novidades  
- **Filtros**:
  - Permite o usuário filtrar entre noticias reais ou falsas
- **Modo claro e escuro**:
  - Permite um melhor conforto visual e customização de acordo com a preferência do usuário

## Documentação  
- Github pages com novo visual e moderno com mais informações sobre o projeto
- README completo para qualquer pessoa conseguir entender do que se trata e como executar o projeto

## Mudanças
- **Search Bar**:
  - Novo método de busca por palavras chave torna a busca mais simples para os usuários
  - Sugestões dos títulos das notícias pela search bar
- **Carregamento das notícias**:
  - Novo método de carregamento das notícias na página por chuncks para evitar carregar todo o banco de dados, agora abre de acordo com a página

## Externos ao site
- **Minerador**:
  - Criação de um minerador para buscar notícias na web para serem classificadas
- **Agente de IA**:
  - Agente de IA implementado no ragflow para classificar as notícias
